# pix-test-react

Minimal Next.js 15 project for testing PixToCode's project-context feature.

## What's inside

- **UI primitives** in `components/ui/`: Button, Card (+sub), Input, Avatar, Badge
- **Domain components** in `components/`: ProductCard, PricingCard, UserCard
- **Design tokens** in `app/globals.css` (CSS variables) and `tailwind.config.ts` (theme.extend)
- **Path aliases** in `tsconfig.json`: `@/*`, `@/components/*`, `@/ui/*`

## Run

```bash
npm install
npm run dev
```
